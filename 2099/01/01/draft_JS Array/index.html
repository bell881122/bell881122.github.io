Cheat Sheet

method 回傳內容 使用方法 簡述
indexOf Number Arr.indexOf(Value) 回傳符合條件第一個元素的 index
indexOf 變體 Number Arr.indexOf(Value,Num) 從指定的 index 開始尋找
findIndex Number Arr.findIndex(Condition) 回傳符合條件第一個元素的 index
find Value Arr.find(Condition) 回傳符合條件的第一個元素
filter Array Arr.filter(Condition) 回傳符合條件的所有元素
some Bool Arr.some(Condition) 部分元素符合即為 true
every Bool Arr.some(Condition) 所有元素符合時為 true
